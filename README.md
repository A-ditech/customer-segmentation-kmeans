# 🛍️ Customer Segmentation using K-Means Clustering

## 📌 Project Overview

This project segments retail customers into different groups based on their **Annual Income** and **Spending Score** using the **K-Means Clustering** algorithm.

The objective is to help businesses understand customer purchasing behavior and create targeted marketing strategies.

---

## 🎯 Objective

- Segment customers into meaningful groups.
- Analyze customer purchasing behavior.
- Generate business insights using machine learning.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📂 Dataset

The dataset contains:

- CustomerID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1–100)

---

## 🚀 Workflow

1. Data Loading
2. Data Cleaning
3. Missing Value Handling
4. Exploratory Data Analysis
5. Feature Scaling
6. Elbow Method
7. K-Means Clustering
8. Customer Segmentation
9. Business Insights

---

## 📈 Results

The optimal number of clusters was determined using the **Elbow Method (K = 5)**.

The model successfully identified five different customer groups based on their purchasing behavior.
